Goal: distinguish "actually done but not flipped" (Closable now) from "spec done; data or human action pending" (Partial-closable) from "fully human-blocking."

Triage criteria:

- **🟢 Closable now**: a locked artifact (D-number / RB-N / phase1/* spec doc) already fulfills the deliverable contract. Status flip is mechanical; no human action required.
- **🟡 Partial-closable**: spec-side IS done in a locked artifact, but the deliverable also requires data collection / human review / implementation. Spec-side closure documented; residual work tracked as B-items.
- **⬜ Human-blocking**: deliverable requires action that cannot be performed autonomously (DBA review, compliance sign-off, source-DB query runs, stakeholder coordination).
